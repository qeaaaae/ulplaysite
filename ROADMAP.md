## Высокий приоритет

### 1. Политика заказа только для авторизованных
- **Проблема:** Гость может попасть в корзину и нажать «Оформление» — нужен редирект на логин
- **Решение:** Редиректить на `/login?redirect=/checkout` или показывать модалку входа; после входа переходить к оформлению
- **Файлы:** `resources/views/orders/checkout.blade.php`, `resources/views/cart/index.blade.php`, `AuthController`, middleware/route для checkout

### 2. Авто-инициализация Tom Select после AJAX
- **Проблема:** При обновлении HTML через AJAX (комментарии, корзина) новые селекты не инициализируются
- **Решение:** Вынести `initTomSelects()` в общую функцию, вызывать после каждой вставки HTML; или `MutationObserver`
- **Файлы:** JS с инициализацией Tom Select (app.js / alpine / layout)

### 3. Уведомление о 30-секундном таймауте комментариев
- **Проблема:** Пользователь не видит причину ошибки до отправки
- **Решение:** После отправки показывать таймер «Следующий комментарий через X сек» или блокировать кнопку с обратным отсчётом
- **Файлы:** Компонент комментариев, `CommentController`

## Средний приоритет

### 4. Контроль доступа к заказам ✅
- **Сделано:** `OrderPolicy` для `view` (владелец, админ, session order_view); `SupportTicketPolicy` для `view` (только владелец)
- **Файлы:** `OrderPolicy`, `SupportTicketPolicy`, `OrderController`, `SupportTicketController`, `Controller` (trait `AuthorizesRequests`)

### 5. Ошибки валидации в модалке тикета
- **Проблема:** Тикет теперь на отдельной странице `/support`, не в модалке — актуальность пункта уточнить
- **Решение:** При ошибках — `redirect()->back()->withInput()->withErrors()`, показывать ошибки в форме
- **Файлы:** `SupportTicketController::store`, `resources/views/tickets/create.blade.php`

### 6. Скрытие картинок с битыми ссылками
- **Проблема:** Внешние URL могут 404
- **Решение:** `onerror="this.style.display='none'"` или заглушка; либо хранить в storage
- **Файлы:** Компонент отображения изображений, Blade-шаблоны

### 7. Eager loading в списках
- **Проблема:** N+1 в `OrderController::index`, `SupportTicketController::myIndex` и т.п.
- **Решение:** Добавить `->with()` для связей
- **Файлы:** Контроллеры, проверить все `paginate()`/`get()`
