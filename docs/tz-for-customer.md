# Техническое задание — ulplay.com

**Интернет-магазин игровых консолей и аксессуаров (Ульяновск)**

---

## 1. Публичная часть сайта

### 1.1 Главная страница
- Слайдер баннеров (управляется из админки)
- Блок категорий (до 6 на планшете, 5 на десктопе, 4 на телефоне)
- Новые поступления (товары)
- Рекомендуемые товары
- Наши услуги
- Последние новости

### 1.2 Каталог товаров
- Список товаров с фильтрами (категория, цена)
- Поиск
- Сортировка (цена, новизна, популярность)
- Пагинация
- Карточка товара: фото, название, цена, описание, «В корзину», «В избранное»

### 1.3 Страница товара
- Галерея изображений
- Название, описание, цена
- Характеристики
- Кнопки «В корзину», «В избранное»
- Блок похожих товаров

### 1.4 Услуги
- Список услуг (ремонт, покупка устройств)
- Карточка услуги: описание, кнопка «Заказать» / добавление в корзину
- Прайс-лист на ремонт (если нужно)
- Форма заявки на ремонт / оценку устройства

### 1.5 Новости
- Лента новостей с пагинацией
- Страница новости: текст, дата, автор
- Комментарии (только для авторизованных)

### 1.6 Корзина
- Список товаров и услуг
- Изменение количества, удаление
- Итоговая сумма
- Переход к оформлению заказа

### 1.7 Оформление заказа
- Контакты (имя, телефон, email, адрес)
- Способ доставки (бесплатно от 3000₽ по Ульяновску)
- Способ оплаты (наличные / карта)
- Подтверждение заказа
- Работа для гостей и авторизованных

### 1.8 Личный кабинет
- Регистрация и вход
- Восстановление пароля
- Редактирование профиля (имя, телефон, email)
- История заказов и детали заказа

### 1.9 Избранное
- Добавление товаров в избранное (сердечко)
- Страница со списком избранных товаров

### 1.10 Информационные страницы
- О нас
- Доставка и оплата
- Контакты (телефон, email, форма обратной связи)

---

## 2. Административная панель

### 2.1 Управление контентом
- **Товары** — добавление, редактирование, удаление (название, описание, цена, фото, категория, наличие)
- **Категории** — дерево категорий, сортировка
- **Услуги** — CRUD (ремонт, покупка)
- **Новости** — CRUD с датой публикации
- **Баннеры** — слайдер на главной (изображение, заголовок, ссылка, порядок)

### 2.2 Управление заказами
- Список заказов
- Просмотр и смена статуса заказа

### 2.3 Управление пользователями
- Список пользователей
- Блокировка / разблокировка

### 2.4 Модерация
- Удаление комментариев к новостям

---

## 3. Функциональные возможности

| Функция | Описание |
|--------|----------|
| **Поиск** | Поиск товаров по названию в шапке и на странице каталога |
| **Корзина** | Работает для гостей (сессия) и авторизованных |
| **Избранное** | Только для авторизованных |
| **Оплата** | Наличные при получении, оплата картой (опционально — онлайн-оплата) |
| **Адаптивность** | Корректная работа на телефонах, планшетах, десктопах |
| **Уведомления** | Toast при добавлении в корзину / избранное; опционально — email о заказах |

---

## 4. Технический стек

- **Backend:** Laravel 12, PHP 8.2+
- **База данных:** MySQL
- **Frontend:** Blade, TailwindCSS, Alpine.js, Swiper.js
- **Сборка:** Vite

| Приоритет | Блок | Содержание |
|-----------|------|------------|
| **Высокий** | Магазин | Каталог, товар, корзина, оформление заказа |
| **Высокий** | Админка | Товары, категории, заказы, баннеры |
| **Средний** | Услуги | Список услуг, заявки |
| **Средний** | ЛК | Регистрация, профиль, история заказов |
| **Средний** | Новости | Лента, статья, комментарии |
| **Низкий** | Избранное, инфо-страницы | О нас, доставка, контакты |
